# CPP-Practicing

A collection of competitive programming solutions in C++, organized by platform and contest.

## Repository Structure

This repository contains solutions to problems from various competitive programming platforms including USACO, AtCoder Beginner Contests (ABC), Codeforces (CF), CSES, LeetCode, and others.

```bash

cd build
cmake ..
make
```
